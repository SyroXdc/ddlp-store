
# DDLP STORE PRO

Modern marketplace starter project.

## Features
- Modern dark UI
- Product marketplace
- Cart system
- WhatsApp checkout
- Admin dashboard
- Product management
- Payment method management
- Contact settings
- Node.js API server

## Run Backend

cd backend
npm install
node server.js

Server:
http://localhost:3000

## Open Website

Open:

frontend/index.html
